# Soccer Blog

A web blog enabling users to view, post, and comment on soccer teams.

## Features

- Role-Based Authentication integrated across both backend and frontend for user management
- Robust backend server built with Flask, using PostgreSQL for reliable data storage
- Flask session with Redis for caching, reducing response time by 12%
- Responsive and intuitive user interface using React and Ant Design
