# Book Notes

A full-stack web application to help users manage their reading history and reflections on books.

## Tech Stack

React, Ant Design, Express.js, Node.js, TypeORM, PostgreSQL, Axios

## Features

- User-friendly interface built with React and Ant Design for responsive design
- Express.js with Node.js backend server, using TypeORM to interact with the database
- PostgreSQL for data storage with CRUD methods for efficient data management and sorting
- Axios integration with a public API enabling dynamic retrieval of book covers based on user-submitted book titles
- JWT-based authentication for user registration and login, ensuring secure access and data protection
