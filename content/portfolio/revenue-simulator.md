# Revenue Simulator Real Estate

A machine learning-powered real estate price prediction and revenue simulation tool for the Kitchener-Waterloo region.

## Tech Stack

Machine Learning, Pandas, Streamlit, PostgreSQL, AWS EC2, Docker

## Features

- Complete data pipeline processing real estate listing data with Random Forest and LightGBM models
- K-Fold cross-validation achieving 96% accuracy in price predictions
- Revenue Estimator integrating a Demand-Price Model for comprehensive revenue forecasting
- Deployed as a Streamlit web application on AWS EC2 with Docker for interactive stakeholder access
- Used as a consulting tool for Conestoga Park Square to optimize revenue and inform strategic decisions
